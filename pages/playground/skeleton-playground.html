CSS Layers (quick glossary)

tokens.css

Design tokens only: colors, gradients, spacing, radii, font sizes, durations, easings.

base.css

Reset/normalize + element defaults and basic layout primitives.

components.css

Global components & utilities: nav, buttons, cards, badges, helpers.

app.css

The app-wide bundle that wires everything together (imports + minor tweaks).
