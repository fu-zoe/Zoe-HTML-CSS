Colors & Backgrounds

Hex/rgb/hsl with alpha, currentColor, color-mix(), gradients (linear/radial/conic), and background layering/size/position tricks.

1) Color formats & alpha

Hex / RGB / HSL

/* Hex (with alpha: #RRGGBBAA) */
.box-hex   { background: #2a6f97; }
.box-hexa  { background: #2a6f97cc; } /* ~80% opacity */

/* rgb() / hsl() with alpha */
.box-rgba  { background: rgba(79,145,180,.75); }
.box-hsla  { background: hsl(204 39% 51% / .75); }
#2a6f97
#2a6f97cc

hsl(... / .75)

currentColor

.icon { color: var(--ocean-700); border: 2px solid currentColor; }

Borders/shadows can reuse the element’s text color.

2) color-mix()

Mix with white/black

.tint  { background: color-mix(in srgb, var(--color-primary) 30%, white); }
.shade { background: color-mix(in srgb, var(--color-primary) 70%, black); }
tint (→ lighter)
shade (→ darker)

On hover

.btn-mix:hover {
  background: color-mix(in srgb, var(--color-primary) 70%, var(--color-surface));
}
Hover me

3) Gradients

Linear

.g-linear { background: linear-gradient(135deg, var(--ocean-500), var(--ocean-700)); }

Radial

.g-radial { background: radial-gradient(circle at 30% 30%, var(--teal-400), transparent 60%); }

Conic

.g-conic { background: conic-gradient(from 0.25turn, var(--ocean-500), var(--teal-500), var(--ocean-500)); }

Text gradient (clip)

.text-grad {
  background: var(--gradient-primary);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
}

Gradient text

4) Multiple backgrounds & layering

Overlay + pattern

.layered {
  background:
    linear-gradient(180deg, rgba(0,0,0,.18), rgba(0,0,0,0)),
    url("data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='60' height='60'><rect width='60' height='60' fill='%23e4f0f8'/><circle cx='30' cy='30' r='6' fill='%236fb0d8'/></svg>");
  background-size: auto, 60px 60px;  /* top layer uses auto; pattern 60px */
  background-repeat: no-repeat, repeat;
  background-position: center, top left;
}

Shorthand & order

/* First listed = topmost layer */
.panel {
  background:
    radial-gradient(circle at 30% 30%, rgba(255,255,255,.25), transparent 50%),
    var(--gradient-section-c);
}

5) Size · Position · Repeat

cover vs contain

.cover  { background-size: cover;  background-position: center; }
.contain{ background-size: contain;background-position: center; background-repeat: no-repeat; }

Repeat & position

.tile { background-repeat: repeat; background-position: top left; }
.no-repeat { background-repeat: no-repeat; background-position: 10% 20%; }
