Absolute & relative URLs, internal anchors, and special types (download / mailto / tel / new tab). Code on the left, live behavior on the right.

Absolute reference

<!-- Absolute URL (full web address) -->
<p>
  Jump to MDN:
  <a href="https://developer.mozilla.org/" target="_blank" rel="noopener">
    Web Docs
  </a>
</p>

Jump to MDN: Web Docs Opens in a new tab.

Relative reference

<!-- Relative URL (same folder) -->
<a href="about.html">About (relative)</a>

About (relative) Demo: this click is simulated (won't 404).

Internal anchors (same page)

<!-- Jump to different areas of the same page -->
<a href="#s1">Jump to Section 1</a>
<a href="#top">Back to top</a>

Jump to Section 1 · Back to top

Special link types

<!-- Download a file -->
<a href="/assets/html-5.png" download>HTML5 icon (file)↓</a>

<!-- Or: data URI (works without any file) -->
<a href="data:image/svg+xml;utf8,<svg xmlns='http://www.w3.org/2000/svg' width='64' height='64'><rect width='64' height='64' fill='%236fb0d8'/><text x='50%' y='55%' dominant-baseline='middle' text-anchor='middle' font-size='18' fill='white'>HTML5</text></svg>"
   download="html5-demo.svg">Download embedded SVG↓</a>

<!-- Email -->
<a href="mailto:random@email.com">Contact me</a>

<!-- Telephone -->
<a href="tel:+123456789">Dial me</a>

<!-- Open in new tab -->
<a href="https://www.google.com/" target="_blank" rel="noopener">Google</a>

<!-- Back to Index -->
<a href="../../index.html">Back to home</a>
