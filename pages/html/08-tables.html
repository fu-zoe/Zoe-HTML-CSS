Chapter 8 Tables

Create Tables

Caption, thead/tbody/tfoot, row/column headers (scope), and cell spanning with rowspan/colspan. Turn on the visual aids to see structure.

Visualization aids

Tip: move the cursor over a column header to highlight that column; hover a row to highlight the row.

Live demo (code ↔ render)

<table>
  <!-- Caption: describes what the table is about -->
  <caption>What table is about (centered by default)</caption>

  <thead>
    <tr>
      <!-- Empty top-left header cell -->
      <th>&nbsp;</th>
      <!-- scope="col" marks column headers -->
      <th scope="col">C1 heading</th>
      <th scope="col">C2 heading</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <!-- scope="row" marks row headers -->
      <th scope="row">R1 heading</th>
      <td>C1R1</td>
      <td>C2R1</td>
    </tr>

    <tr>
      <th scope="row">R2 heading</th>
      <td>C1R2</td>
      <td>C2R2</td>
    </tr>

    <!-- This row merges two columns: colspan -->
    <tr>
      <th scope="row">R3 heading</th>
      <td colspan="2">Taking up two columns</td>
    </tr>

    <!-- This row and the next row are merged in the first data column -->
    <tr>
      <th scope="row">R4 heading</th>
      <td rowspan="2">Taking up two rows</td>
      <td>row1</td>
    </tr>
    <tr>
      <th scope="row">R5 heading</th>
      <td>row2</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan="3">Table footer</td>
    </tr>
  </tfoot>
</table>
What table is about (centered by default)
  C1 heading C2 heading
R1 heading C1R1 C2R1
R2 heading C1R2 C2R2
R3 heading Taking up two columns
R4 heading Taking up two rows row1
R5 heading row2
Table footer

Hover a column header to highlight the whole column; hover a row anywhere to highlight the row. Spanned cells show badges when enabled.
