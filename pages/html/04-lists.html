Chapter 4 Lists

List Types

Ordered, unordered, and description lists — code on the left, render on the right (stacks on small screens).

List rules (quick)

  • Do not wrap a list in <p>; the list is a block by itself.
  • <li> may contain paragraphs or other blocks if needed.
  • <dl> pairs use <dt> (term) + <dd> (definition).

Ordered list (<ol>)

<!-- Ordered List -->
<!-- Lists should not be wrapped in <p>; but <li> may contain blocks -->
<ol>
  <li><p>point1 inside a paragraph</p></li>  <!-- legal but uncommon -->
  <li>point2</li>
  <li>point3</li>
</ol>

  1. point1 inside a paragraph

  2. point2
  3. point3

Unordered list (<ul>)

<!-- Unordered List -->
<ul>
  <li><p>point1 inside a paragraph</p></li>  <!-- same note as above -->
  <li>point2</li>
  <li>point3</li>
</ul>

  • point1 inside a paragraph

  • point2
  • point3

Description list (<dl>)

<!-- Description List -->
<dl>
  <dt>description term1</dt>  <!-- term -->
  <dd>description detail1</dd> <!-- definition -->
  <dt>description term2</dt>
  <dd>description detail2</dd>
</dl>
description term1
description detail1
description term2
description detail2
